# safeguarding

This Safeguarding practices webpage is a landing page for people who are concerned on that subject matter and want to understand more. It explains what Safegurding means, what to look for and how to react. Maybe they need help or need to report a situation.

## Features

1. Header image and statement
2. Link to section buttons
3. "What is Safeguarding" section
4. "Sign to watch for" section
5. "Action steps" section, as bullet points
6. Contact form
